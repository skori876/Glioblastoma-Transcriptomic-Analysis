 Glioblastoma Transcriptomic Analysis (GSE4290)

Project Overview

This project performs differential gene expression and functional enrichment analysis on glioblastoma (GBM) and normal brain tissue samples using the GEO dataset GSE4290.

The workflow includes:

* Downloading and preprocessing microarray expression data from GEO
* Differential expression analysis using the Limma package
* Identification of significantly upregulated and downregulated genes
* Gene Ontology (GO) enrichment analysis
* KEGG pathway enrichment analysis
* Export of differential expression and enrichment results

 Dataset

* GEO Accession: GSE4290
* Platform: GPL570 (Affymetrix Human Genome U133 Plus 2.0 Array)
* Samples:

  * Glioblastoma (GBM): 77
  * Normal Brain Tissue: 23
